##1. Pergunta e Resposta

Um jogo de quiz com perguntas de múltipla escolha.

Como funciona:

O jogador responde 5 perguntas
Cada pergunta possui 4 alternativas
O programa verifica se a resposta está correta
Ao final, é exibida a pontuação total

Conceitos utilizados:

Estruturas condicionais (if)
Variável acumuladora (pontuação)
Entrada de dados com scanf

##2. Cobra na Caixa

Um jogo para dois jogadores baseado em sorte e estratégia.

Como funciona:

Existem 5 caixas fechadas
Uma contém a saída (vitória)
Outra contém uma cobra (derrota)
Os jogadores escolhem caixas alternadamente
O jogo termina quando alguém encontra a cobra ou a saída

Conceitos utilizados:

Vetores (controle das caixas abertas)
Números aleatórios (rand)
Estrutura de repetição (while)
Controle de turno

##3. Gousmas War

Um jogo estratégico entre dois jogadores com controle de estado.

Como funciona:

Cada jogador possui até 2 gousmas
Cada gousma possui um valor de “fúria”
O jogador pode escolher entre:
Atacar: aumenta a fúria de uma gousma inimiga
Dividir: divide a fúria entre duas gousmas
Se a fúria ultrapassar 5, a gousma é destruída
O jogo termina quando um jogador perde todas as suas gousmas

Conceitos utilizados:

Controle de estado com variáveis
Estruturas condicionais complexas
Validação de ações
Alternância de turnos
Lógica de jogo
